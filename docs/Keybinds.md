# Keybinds

Keyboard shortcuts for Calendaria widgets and controls.

---

## Toggle Keybinds

| Keybind            | Action                   | Default |
| ------------------ | ------------------------ | ------- |
| Toggle BigCal      | Opens/closes BigCal      | Unbound |
| Toggle MiniCal     | Opens/closes MiniCal     | Unbound |
| Toggle HUD         | Opens/closes HUD         | Alt+C   |
| Toggle Time Keeper | Opens/closes Time Keeper | Unbound |
| Toggle Sun Dial    | Opens/closes Sun Dial    | Unbound |
| Toggle Chronicle   | Opens/closes Chronicle   | Unbound |
| Toggle Note Viewer | Opens/closes Note Viewer | Unbound |
| Toggle Stop Watch  | Opens/closes Stop Watch  | Unbound |

---

## Stop Watch Controls

| Keybind                | Action                           | Default |
| ---------------------- | -------------------------------- | ------- |
| Start/Pause Stop Watch | Toggles stop watch running state | Unbound |
| Reset Stop Watch       | Resets stop watch to zero        | Unbound |

---

## Configuring Keybinds

Keybinds are configured via Foundry's core settings:

1. Open Game Settings (gear icon)
2. Click "Configure Controls"
3. Scroll to "Calendaria" section
4. Click the key field to set a new binding
5. Press the desired key combination

---

## Notes

- Only Toggle HUD has a default binding (Alt+C)
- All other keybinds are unbound by default to avoid conflicts with other modules
