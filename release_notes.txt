## Video Walkthrough

A walkthrough of most major features, graciously created by the BaileyWiki team: https://youtu.be/yrl0Wh54ypQ

## New Features

- `/enrichers` command creates a reference journal with live examples for all 51 enrichers
- "Show Time" toggle to hide time display and controls on MiniCal
- Time increment selection persists across reloads for all widgets

## Bug Fixes

- Fixed rest advancing time once per player instead of once per rest
- Fixed theme preset switching reverting to previous theme
- Fixed season enricher always returning first season
- Fixed moon/zone enrichers calling non-existent methods
- Fixed settings panel sidebar overflowing at large font sizes
- Fixed Note Viewer FAB jumping with scrollbar changes
- Fixed missing Sunrise/Sunset localization for approximate time
- Fixed HUD not restoring increment selection on fresh worlds

## Changes

- Replaced single "Custom" theme with per-preset Customize workflow
- Theme editor redesigned -- removed Location column, compact headers, preset hint
- Settings panel height increased to 860px
