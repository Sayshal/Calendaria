## New Features

- Added CALENDARIA.api.setActiveCalendar(id, { useCalendarDefaults }) for switching calendars from scripts
- Chronicle enricher links accept a categories="a,b" parameter to open the timeline pre-filtered without altering the saved filter
- Chronicle Timeline toolbar gains a category filter popover with toggle options, selected count badge, and clear action; selection persists per client
- Climate editor adds an inline pencil-toggle alias editor on each zone temperature row, with the indicator only lighting for real overrides
- Locked-range chronicle views now show the date range below the toolbar buttons so actions stay on a single row
- New Automatic rest mode (default) advances time using the system-reported duration; the old Fixed mode is now called Custom and accepts 0 hours to skip advancement
- New action enrichers for time and weather: advanceToTime, advanceInterval, advanceToPreset, setWeather, setTime, jumpToDate, and toggleClock, all permission-gated and socket-routed for non-GMs
- New GM-only Print / Export buttons in Settings -> Module render the active month or full year as a printable (PDF-saveable) layout with festivals and moon phases
- New worlds seed display formats from the active calendar's dateFormats, so dates use the calendar's defaults by default
- Settings show a "<Calendar> Default" preset that follows calendar switches, and note viewer/filter/date picker plus cinematic keyframes and overlay now route through configurable format locations
- Weather zones can define per-zone season aliases (name, abbreviation, icon, color) applied across HUD, calendar UI, chronicle, cinematics, enrichers, and chat

## Bug Fixes

- Chronicle date enrichers no longer crash when a non-numeric token (such as a month name from a different calendar) is supplied
- Custom Weekdays no longer return phantom days on Foundry v14, weekday names may be left blank, and the settings panel renders again
- Fixed weekday math drifting after intercalary festivals and labeled intercalary-month cells in calendar grids with the month name (BigCal) or first letter (MiniCal) plus tooltip
- Moon brightness contribution edits now persist when saving the calendar editor
- Moon phases now stay aligned across HUD, sun-dial, enrichers, and darkness lighting by honoring yearZero, referencePhase, cycleDayAdjust, and reset anchors consistently with the mini and big calendars
- Multi-day reminders now fire on the correct day for offsets greater than 24 hours, with the offset cap removed and check frequency tightened to 1 game-minute
- Weather presets with relative tempMin/tempMax modifiers (e.g. "61+") now fire correctly and extend the temperature range outside the season bounds instead of being clamped back
