## New Features

- Added `to="event"` keyword to the `cal.countup` enricher so countup expressions embedded in a calendar note page automatically resolve against that event's date, no hardcoded target required. Tooltips now correctly say "X days until" for future static dates and "X days since" for past or live dates.
- French (Français) and Spanish (Español) translations are now available in Foundry's language picker, so users can run Calendaria in those locales.
- Multi-day note bars in BigCal now scale to the note's start and end times of day, so a note ending at 1pm visually stops mid-afternoon on its last day instead of filling the full square, with all-day notes and interior weeks still spanning their days fully.
- The `cal.countup` enricher's `to=` parameter now accepts a calendar note name (case-insensitive, exact match preferred over substring), so you can write `[[cal.countup 1 Mirtul 1490 to="High Harvestide"]]` to count to a named event instead of hardcoding its date. Only notes you can observe are matched.
- Today's date is now visibly highlighted in MiniCal and BigCal with a distinct background tint and day-number colour, and stays readable when also selected. Theme presets (Arcane, Frost, Minimalist, Sakura, Slate, Neon, Parchment, Solarized, Steampunk, Verdant) were recoloured so the primary and "today" accents no longer clash or share hues.

## Bug Fixes

- Calendaria's ambient weather audio now fades out when your token enters a Foundry region with the core Suppress Weather behavior, and restores when you leave, so indoor and sheltered areas stay quiet without manual muting.
- Cinematic event cards now respect note permissions, so players no longer see hidden GM notes flash by during the cinematic preview.
- Cinematic event cards now stay in sync with the day shown on the clock, no longer bleeding into the following day or vanishing after a short timeout.
- Fixed events disappearing from cinematics on calendars with a non-zero year offset (e.g. Gregorian), so festival and note cards now appear on the correct days.
- In augment chat-timestamp mode, the in-game date now appears as a small italic banner at the bottom-right of each chat message instead of replacing Foundry's "Xs ago" timestamp, so the delete and context icons are no longer squashed.
- Players who have hidden the calendar HUD no longer have it reappear when the GM switches scenes.
- Pressing Today in MiniCal now populates the notes pane for today, and opening MiniCal from BigCal auto-opens today's notes when the auto-open setting is enabled.
