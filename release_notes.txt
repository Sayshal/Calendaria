## New Features
- PF2E "Rest for the Night" now advances world time (8 hours or to sunrise)
- Starfinder 2E world clock sync — Golarion calendar syncs with SF2E's built-in clock
- Per-scene FX Top-Down Override (Default / Top-Down / Side View) in scene config
- Climate zone preset enabled, temperature, and inertia settings now save per-season

## Bug Fixes
- Empty translation strings in fr, it, pl, and tr language files no longer cause blank UI
- Climate editor no longer loses inertia values on re-render
- Localization guard prevents undefined calendar names when using incomplete language packs
