## New Features
- [#465] Weather Probability dialog — view the effective weight breakdown for each preset by zone and season. Access from Settings, Climate Editor, Weather Picker, or `/weatherprob` chat command
- [#465] Season-based weight overrides — zone presets now use per-season weight overrides instead of a single flat weight. Select a season in the Climate Editor to set weights per preset with support for relative modifiers (`10+`, `5-`)

- [#468] All notes appeared on the 1st of the month after migration
