## Bug Fixes

- [#683] Custom format strings now accept `{literal}` in addition to `[literal]`, allowing literal text inside `[[ ]]` enrichers without bracket collisions
- [#685] Moon phase anchors on randomized moons with weighted phases now land inside the intended phase slice instead of drifting into an adjacent phase
- A malformed custom calendar or default override no longer halts module startup; the broken calendar is skipped and logged, letting the rest of the module load normally
